我们的项目

在编程小白社，我们致力于发展多个创新和教育项目，帮助社区成员学习、成长，并分享他们的知识。

  • SZU公共校历及通讯录（维护中）GitHub 仓库

    SZU公共校历及通讯录是一个为师生提供重要日期和联系信息的重要资源。当前，这个工具正在维护中，以确保所有信息都是最新和最准确的。公共校历包括了学期开始和结束的日期、假期、重要的学术和社交活动日期等，帮助学生和教职员工更好地规划他们的学术和个人生活。通讯录则提供了校内各部门和关键职员的联系方式，便于快速沟通和信息查询。维护期间，我们致力于改进用户体验并更新数据，以便在重新上线后，能提供一个更加完善和用户友好的服务。感谢大家的理解和耐心等待。

  • 社群专属Wiki（筹备中）测试站点

    社群专属Wiki（筹备中）是一个旨在为社群成员提供专门知识和资源的在线平台。这个Wiki将包括从基础教程到高级编程技巧的广泛内容，专为社群内的成员定制，以支持他们的学习和成长。目前，该项目正在筹备阶段，我们正努力构建一个内容丰富、易于导航和用户友好的界面。这个平台将使社群成员能够不仅查阅信息，还可以互相协作，共同创建和维护内容。通过这种方式，社群专属Wiki旨在成为一个自我增长和知识共享的强大工具，促进社群内的互动和发展。

  • 社群官网（筹备中）GitHub 仓库

    社群官网（筹备中）是为我们的社区成员提供一个中心化的在线访问点，旨在集成所有相关资源、活动和通讯工具。目前，这个网站正在筹备阶段，我们的团队正在努力设计一个既美观又功能全面的平台。该官网计划包括社区新闻、活动日历、项目展示、成员互动区域及更多功能，以支持社区成员的交流和协作。此外，它也将提供对外的介绍，帮助感兴趣的新成员了解社区的文化和价值。我们期望这个官网成为社区生活的核心，促进成员间的连结和成长。

  • HoopMates约球小程序（筹备中） GitHub 仓库

    HoopMates（约球小程序）是一个专为篮球爱好者设计的移动应用程序，旨在帮助用户轻松找到附近的球友和球场，安排篮球活动。这个小程序通过地理定位服务显示最近的篮球场地，用户可以查看场地详情、预约场地以及加入即将开始的篮球比赛。用户还可以创建自己的篮球活动，邀请其他用户参加。
    HoopMates的界面友好，操作简便，不仅使球友们能快速组织或参加篮球活动，还能根据技能水平和兴趣匹配合适的球友，增强比赛的竞争性和乐趣。此外，HoopMates还提供社交功能，如用户评分、活动留言板和篮球技巧分享，旨在建立一个活跃且互助的篮球社区。无论你是业余爱好者还是资深球员，HoopMates都能为你提供一个连接篮球世界的窗口。
